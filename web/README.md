# 🖥️ VoluntaFlow - Frontend

Esta é a interface do usuário do sistema VoluntaFlow, focada em acessibilidade, performance e design responsivo.

## ✨ Funcionalidades

* **Dashboard de Impacto:** Estatísticas reais sobre desastres no Brasil.
* **Manual de Sobrevivência Dinâmico:** Interface em abas (Tabs) para acesso rápido a informações de emergência, pós-enchente e prevenção.
* **Mapa de Risco Interativo:** Visualização geográfica das áreas de atenção.
* **Formulário Inteligente:** Cadastro de voluntários com máscara de telefone em tempo real (Regex) e validação de campos.

## 🛠️ Tecnologias

* **Next.js 14** (App Router)
* **Tailwind CSS** (Estilização Utilitária)
* **Lucide React** (Pacote de Ícones)
* **React State Management** (Controle de formulários e abas)

## 🏃 Como rodar localmente

1.  Acesse a pasta: `cd web`
2.  Instale as dependências: `npm install`
3.  Inicie o servidor: `npm run dev`
4.  Acesse: `http://localhost:3000`

## 🎨 Padrões de Design
O projeto segue uma estética corporativa e limpa, utilizando tons de azul para confiança e vermelho/laranja para alertas críticos, evitando o uso de emojis em áreas técnicas para manter o profissionalismo.
